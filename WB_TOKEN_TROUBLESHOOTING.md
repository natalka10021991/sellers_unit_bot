# Решение проблемы с WB API токеном

## Ошибка: "token scope not allowed"

Эта ошибка означает, что токен не имеет прав для доступа к категориям.

### Возможные причины:

1. **Токен создан не для категории "Контент"**
   - Нужно создать токен специально для категории "Контент" (Content)
   - Другие категории токенов не работают для получения категорий товаров

2. **Токен истек или неактивен**
   - Проверь срок действия токена в личном кабинете WB

3. **Неправильный формат токена**
   - Токен должен быть JWT токеном (длинная строка)
   - Убедись, что скопирован полностью

### Как проверить токен:

1. Зайди на https://dev.wildberries.ru/
2. Перейди в раздел "API токены"
3. Проверь:
   - ✅ Токен для категории **"Контент"** (Content)
   - ✅ Токен активен
   - ✅ Срок действия не истек

### Как создать правильный токен:

1. Зайди на https://dev.wildberries.ru/
2. Перейди в "API токены" → "Создать токен"
3. Выбери категорию: **"Контент"** (Content)
4. Скопируй токен
5. Добавь в `.env`:
   ```env
   WB_API_TOKEN=твой_новый_токен
   ```
6. Перезапусти бота

### Альтернативное решение:

Если токен для "Контент" недоступен, можно:
- Использовать статический список популярных категорий
- Запрашивать комиссию у пользователя вручную
- Использовать примерные значения комиссий по категориям
